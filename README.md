# 📊 Amazon Sales Analysis (SQL + Power BI)

## 📌 Project Overview

This project focuses on analyzing Amazon sales data to uncover key business insights using SQL for data querying and Power BI for visualization.

The goal is to transform raw data into meaningful insights related to revenue, customer regions, product categories, and sales performance.

---

## ⚙️ Tools & Technologies

* SQL (MySQL)
* Power BI
* Data Cleaning & Transformation

## 🔍 Business Questions Solved

* What is the total revenue generated?
* Which customer region generates the highest revenue?
* What is the average quantity sold per order?
* Which product categories have the highest ratings?
* How does sales distribution vary across regions?

## 📈 Key Insights

* North America contributes the highest share of total revenue.
* Electronics and Home categories show consistently high ratings.
* Sales distribution varies significantly across regions.
* Quantity sold trends indicate region-based demand differences.
